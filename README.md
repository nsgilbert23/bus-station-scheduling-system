# Bus Station Scheduling System

## Description
The City Bus Station Scheduling and Boarding System is designed to manage bus departures and passenger boarding efficiently using data structures. The system utilizes a priority queue to schedule buses based on departure time and urgency, ensuring that buses with higher priority depart first. A queue data structure is used to manage passengers waiting to board the buses in a first-in-first-out (FIFO) manner

## Data Structures Used
- Priority Queue (Bus Scheduling)
- Queue (Passenger Boarding)

## Features
- Add buses
- Add passengers
- Start boarding
- Delay buses
- Cancel buses

C++
